# PracticeCode

Collection of Python practice scripts and notebooks for language and data analysis basics.

## Contents

* `number.py`: Python number and string basics examples
* `numpy_examples.py`: NumPy array operations and list vs NumPy performance comparison
* `pandas.py`: Minimal pandas import starter file
* `shoe_sales_analysis_mean_median.ipynb`: Notebook for sales analysis
* `ShoeAnalysis.ipynb`: Additional notebook analysis
* `shoe_sales.csv`: Sample dataset

## Prerequisites

* Python 3.10+
* Dependencies from `requirements.txt`

## Setup

```bash
cd PracticeCode
pip install -r requirements.txt
```

## Run Script Examples

```bash
cd PracticeCode
python number.py
python numpy_examples.py
```
