# CodeApp

This folder contains C# console-based practice code and algorithm examples.

## Contents

* `Program.cs`: Entry point with sample algorithm logic (`isSubSequence`).
* `ArrayConcatenation.cs`: Array-related practice implementation.
* `MedianOfTwoSortedArrays.cs`: Median-of-two-sorted-arrays practice implementation.
* `example.cs`: Additional experimentation file.

## Prerequisites

* .NET SDK 8.0 or later

## Run

```bash
cd CodeApp
dotnet run
```

## Build

```bash
cd CodeApp
dotnet build
```
